# Crash-Inspired Runner

Этот проект содержит небольшую 2D-игру, вдохновлённую серией Crash Bandicoot.
Управляйте героем, который несётся вперёд, разбивает ящики и уклоняется от
опасностей. Игра написана на Python с использованием библиотеки Pygame.

## Запуск

1. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Запустите игру:

   ```bash
   python game.py
   ```

## Управление

- **Пробел** — прыжок.
- **Левый Ctrl** — вращение, позволяющее ломать ящики и уничтожать некоторые
  препятствия.
- **Esc** — выход из игры.

## Игровой процесс

Избегайте нитро-ящиков, собирайте маски Aku-Aku для кратковременной защиты и
разбивайте обычные ящики, чтобы набирать очки. Когда у вас закончатся жизни,
появится экран Game Over — нажмите любую клавишу, чтобы попробовать снова.
